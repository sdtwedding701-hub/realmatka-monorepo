Native branding assets live here.

Current files:

- `icon.png`
- `adaptive-icon.png`
- `splash.png`

You can override them in `.env.local` or production env:

- `EXPO_PUBLIC_APP_ICON=./assets/branding/icon.png`
- `EXPO_PUBLIC_ADAPTIVE_ICON=./assets/branding/adaptive-icon.png`
- `EXPO_PUBLIC_SPLASH_IMAGE=./assets/branding/splash.png`
